What is XMLHttpRequest

  • XMLHttpRequest :
    자바스크립트로 HTTP Request를 다루는 브라우저 내장형 객체이다.
    파일 업로드/다운로드, 진행상황 추적 등을 다룰 수 있다.

웹서버에서 오는 데이터 요청을 다루는 객체이다.
: 페이지 리로드 없이 웹 페이지를 업데이트할 수 있다.
: 서버에 데이터를 요청하고 페이지를 리로드할 수 있다.
: 서버에서 데이터를 받아 페이지를 리로드 할 수 있다.
: 백그라운드에서 서버로 데이터를 보낼 수 있다.

{example}
let xhttp = new XMLHttpRequest(); // creates an XMLHttpRequest object (XMLHttpRequest 객체를 생성한다.)

xhttp.onreadystatechange = function() { // onreadystatechange property : specifies a function to be executed every time the status of the XMLHttpRequest object changes
// (onreadystatechange 프로퍼티는 XMLHttpRequest 객체의 상태가 변경될 때마다 수행될 함수를 정의한다.)
if (this.readyState == 4 && this.status == 200) { // the response is ready (response가 준비되었다.)
// Typical action to be performed when the document is ready:
document.getElementById(“demo”).innerHTML = xhttp.responseText; // responseText property : returns the server response as a text string.
// (responseText 프로퍼티가 서버 응답을 텍스트 형태로 반환한다.)
}
};
xhttp.open(“GET”, “filename”, true);
xhttp.send();
{example}

Unit Test AMD 모듈에 Karma, Mocha 그리고 RequireJS 설정하기

원본: https://medium.com/@BoweiHan/setup-for-testing-amd-modules-with-karma-requirejs-and-mocha-2be3931c6a72

웹 어플리케이션의 복잡도가 증가함에 따라 Front-end 기능을 수동적으로 테스팅하는 것은 더 이상 합당한 정도의 신뢰를 제공하지는 못 한다.
새 기능을 개발하는 데에 들이는 시간은 이제 점점 늘어나는 회귀 문제()를 수정하는 데에 사용되었다. (* Software regression : https://en.wikipedia.org/wiki/Software_regression)

다른 많은 회사와 마찬가지로 나의 회사도 매일매일 개발에 고군분투하고 있다. 이것은 새롭게 개발된 기능들이 프로덕트로 출시되기 전에 겨우 하루도 되지 않는 시간동안 테스트 될 것이라는 것을 의미한다.
